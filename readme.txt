IpLoc - PHP library to detect IP location by masks database

Features:
 * Import IP locations masks from http://www.maxmind.com/app/geolitecity CSV to database
 * Get coordinates, country and city by IP from local database
 * Support MySQL and PostgreSQL
 * Use compact database format < 150mb total data size
 * Works very quickly < 5ms per request
